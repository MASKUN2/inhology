inhology — UI 화면 목록

실제 앱의 12개 화면을 정적 HTML로 재현한 와이어프레임입니다. 외부 리소스 없이 독립 실행되며, 최소한의 CSS(하나의 회색 테두리 · 시스템 폰트 · 색상 없음)로 구조와 문구만 보여줍니다. OS 설정에 따라 라이트/다크가 자동 전환됩니다.

공개 화면 (Public)

  1. 홈 / 글 목록 — /
  2. 글 상세 — /posts/{slug}
  3. 카테고리 아카이브 — /categories/{slug}
  4. 태그 아카이브 — /tags/{slug}
  5. 시리즈 목록 — /series
  6. 시리즈 상세 — /series/{slug}

관리자 화면 (Admin)

  1. 관리자 로그인 — /admin/login
  2. 글 관리 대시보드 — /admin
  3. 새 글 쓰기 — /admin/new
  4. 글 수정 — /admin/posts/{id}/edit
  5. 댓글 관리 — /admin/comments
  6. 시리즈 관리 — /admin/series
