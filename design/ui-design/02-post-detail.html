#MinIO #Next.js
시리즈 · 홈랩 구축기 · 2편

커버 이미지 필드를 걷어내고, 이미지를 본문 안에서만 다루기로 했습니다. 붙여넣기 · 드래그 · 파일 선택 중 무엇으로 올려도 커서 위치에 ![](…)가 자동으로 삽입됩니다.

업로드 경로

매직바이트로 타입을 검사하고(PNG/JPEG/WebP/GIF, SVG 거부), 내용 해시를 키로 삼아 MinIO에 PUT합니다.

POST /admin/api/upload  →  { url: "/images/<sha256>.png" }

객체 저장소 주소는 리더에게 노출되지 않고, 이미지는 사이트 도메인 뒤에서 스트리밍됩니다.
